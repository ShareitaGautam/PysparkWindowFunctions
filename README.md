# PysparkWindowFunctions
This project demonstrates basic PySpark operations including creating DataFrames, applying window functions (row_number, rank, dense_rank, lag, lead), working with date functions, and handling timestamp data.

## Project Overview
This project is a simple PySpark practice script created while learning Data Engineering concepts.  
The script demonstrates how to work with Spark DataFrames and apply important PySpark functions used in real data processing tasks.

The main topics covered are:

- Creating and inspecting DataFrames
- Window Functions
- Date Functions
- Timestamp Functions
